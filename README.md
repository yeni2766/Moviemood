# 🎬 Movie Mood

A movie recommendation web app that suggests films based on your current mood.

Users can select a mood and instantly get a list of movies fetched from an external API, presented in an interactive slideshow format.

⸻

# 🚀 Features
	•	🎭 Mood-based movie selection
	•	🔄 Dynamic API fetching (TMDB)
	•	⏳ Loading spinner for better UX
	•	⚠️ Error handling for failed requests
	•	🎞️ Slideshow navigation (Next / Prev)
	•	🎨 Responsive UI with Tailwind CSS
	•	💡 Type-safe development using TypeScript

⸻

# 🛠️ Tech Stack
	•	React
	•	TypeScript
	•	Tailwind CSS
	•	React Router
	•	TMDB API

⸻

# 📸 How It Works
	1.	Select a mood (e.g. Funny, Romantic, Thriller)
	2.	App maps mood → movie genre
	3.	Fetches movies from TMDB API
	4.	Displays results in a slideshow format
